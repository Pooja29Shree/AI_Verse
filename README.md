# Arumbu – Agentic AI Career Development Copilot

Arumbu is an **Agentic AI–based career guidance platform** that helps students and early-career professionals identify suitable roles, understand skill gaps, and follow a **personalized, gamified roadmap** to job readiness.

## Agents Overview

* **Login Module**
  Secure authentication and user identity management.

* **Psychology Agent**
  Analyzes user responses to infer interests and psychological traits.

* **Web Scraping Agent**
  Collects real-world career roles, skills, and requirements.

* **Role Assignment Agent**
  Matches user traits with roles and identifies skill gaps.

* **Roadmap Generator Agent**
  Creates a step-by-step, gamified career roadmap.

---

## Key Features

* Psychology-based career analysis
* Real job-market driven role suggestions
* Skill gap identification
* Gamified, milestone-based roadmaps
* Progress tracking and dynamic updates
* Offline-capable career knowledge base

---

## Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js (Express)
* **Database:** MongoDB
* **Auth:** Firebase
* **NLP:** NLTK
* **Web Scraping:** BeautifulSoup

## Why Agentic AI?

* Multiple specialized agents
* Persistent memory
* Think → Plan → Act → Learn loop
* Adapts continuously with user progress
